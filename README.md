# bun

Tool for DC/OS diagnostic bundle analysis

## Installation

Download binaries from the [release page](https://github.com/adyatlov/bun/releases) or install from source:

```bash
$ go get github.com/adyatlov/bun/bun
```

## Usage

```bash
$ unzip bundle.zip
$ cd bundle
$ gunzip -r ./
$ bun
```
